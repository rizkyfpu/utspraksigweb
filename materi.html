Materi Pembelajaran

How Internet Works

Client adalah perangkat yang melakukan akses terhadap website (orang/manusia yang mengakses juga dapat disebut client)


Server adalah perangkat keras maupun lunak komputer yang berfungsi untuk menyimpan halaman web pada alamat tertentu


Komunikasi antara klien dan server dilakukan dengan bahasa dan protokol tertentu, yaitu internet.

HMTL dan CSS

HTML (Hyper Text Markup Languages) merupakan salah satu bahasa scripting yang sering digunakan dalam menyusun sebuah website. HTML merupakan bahasa markup yang mendefinisikan struktur informasi dengan menggunakan variasi tags dan atribut. Tag dan atribut didesain untuk menampilkan teks dan informasi lainnya pada layar dan menyediakan hyperlinks ke dokumen web lainnya.

CSS (Cascading Style Sheets) merupakan pelengkap dari penyusunan sebuah website. CSS digunakan untuk memberikan styling dan mengatur layout dari websiet yang dibuat. Kenapa harus menggunakan CSS? Karena pemilihan font, pewarnaan font, spacing, dan styling layout pada HTML sangat terbatas.

JavaScript

Bahasa pemrograman memerintahkan bagian tertentu dari robot untuk melakukan suatu aksi, seperti bergerak, berputar, dan lain-lain. Javascript, Bahasa pemrograman web, memerintahkan elemen tertentu (misalnya button) pada halaman web untuk melakukan suatu aksi.

Bootstrap Framework

Boostrap sebagai salah satu framework javascript yang paling banyak digunakan. Bootstrap menyediakan komponen yang sudah 'dibungkus' untuk memudahkan pembangunan web menggunakan HTML, CSS dan JS Web yang dibangun dengan Bootstrap bersifat Responsive, artinya dapat dengan mudah menyesuaikan ukuran pada browser desktop atau mobile

LeafletJs

LeafletJS merupakan library atau kumpulan fungsi berbasis Javascript yang digunakan untuk menampilkan peta interaktif pada halaman web. Leaflet menyediakan Map API (Application Programming Interface) yang memudahkan web developer untuk menampilkan peta pada sebuah halaman web. Pengguna peta juga dapat berinteraksi dengan menggunakan fungsi telah disediakan oleh Leaflet.

Mobile GIS

Mobile GIS memungkinkan pengguna untuk melakukan pengukuran dan akses data spasial melalui smarthphone. Salah suatu aplikasi yang mendukung adanya Mobile GIS adalah Mergin Apps. aplikasi Mergin Maps: QGIS in pocket sebagai perangkat berbasis mobile, QGIS sebagai perangkat desktop serta Mergin sebagai aplikasi GIS berbasis online dapat digunakan sebagai sarana untuk pengumpulan data dari lapangan secara kolaboratif. Pengumpulan data menggunakan kombinasi ketiga perangkat tersebut dapat digunakan dalam kegiatan survei lapangan, misalnya terkait dengan survei pelanggaran tata ruang, pemetaan aset, pemetaan tempat makan, pemetaan ATM maupun monitoring pembangunan.
